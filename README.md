## Keybindings

| Keybinding                    | Action                              |
| ----------------------------- | ----------------------------------- |
| `B`                           | Toggles the brush tool              |
| `C`                           | Toggles the clip tool               |
| `Ctrl-Enter`                  | Toggles the clip area               |
| `Alt + LMB Drag`              | Move a brush along the Z axis       |
| `LMB Drag` (No tool selected) | Move a brush along the X and Y axis |
